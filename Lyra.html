UE5 Lyra's Expanse map - Masters Project

Created using C++ and Blueprints in Unreal Engine 5, this project improved UE5 Lyra’s Expanse game map by enhancing gameplay with increased variation and difficulty, through weapon stat adjustments and added environmental challenges.

These changes included:
• 3 different evolution variants of each weapon with different values.
• Environmental challenges such as randomly spawning explosives and poisonous fog clouds.
• Changes to the skybox and lighting of the map.

This project was graded with a Merit at University.
